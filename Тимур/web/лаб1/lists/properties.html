Список стилей

  1. font-family: Семейство шрифтов.
  2. background-color: Цвет фона.
  3. padding: Внутренние отступы.
  4. border-radius: Радиус границы.
  5. line-height: Высота строки.
  6. color: Цвет текста.
  7. text-align: Выравнивание текста.
  8. float: Обтекание элементов.
  9. margin: Внешние отступы.
  10. font-size: Размер шрифта.
  11. list-style-type: Тип маркера списка.
  12. content: Содержимое псевдоэлемента.
  13. background-image: Фоновое изображение.
  14. background-size: Размер фонового изображения.
  15. border-collapse: Способ отображения границ таблицы.
  16. text-transform: Преобразование текста.
  17. border: Граница элемента.
  18. display: Способ отображения элемента.
